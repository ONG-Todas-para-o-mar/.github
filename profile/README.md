# 🏄‍♂️ Surfando no Futuro - ONG TPM (Todas Para o Mar)

Bem-vindo à página central do projeto **Surfando no Futuro**! Este é um projeto de modernização digital desenvolvido para a ONG TPM - Todas Para o Mar, inserido no contexto do Projeto de Curricularização da Extensão (PCE) da Universidade Católica de Santos.

O objetivo principal é reformular a presença digital da ONG, criando uma plataforma web mais interativa, acessível, responsiva e eficiente para melhorar a comunicação com a comunidade, sócios e apoiadores.

---

## 📂 Estrutura do Projeto

Para garantir uma arquitetura em camadas e organizada, o ecossistema do nosso sistema está dividido em dois repositórios principais dentro desta organização:

- **🖥️ [Repositório Frontend](https://github.com/ONG-Todas-para-o-mar/ong-tpm-frontend):** Interface do usuário desenvolvida em React.js, JavaScript e Tailwind CSS.
- **⚙️ [Repositório Backend](https://github.com/ONG-Todas-para-o-mar/ong-tpm-backend):** Lógica de negócios, autenticação e APIs desenvolvidas em Python com Django, integradas ao banco de dados PostgreSQL.

---

## ✨ Funcionalidades Principais

O sistema foi modelado para atender a dois perfis de usuários (Visitantes e Sócios), contando com os seguintes recursos:

- **🔑 Autenticação Segura:** Sistema de cadastro e login de usuários com autenticação via JWT.
- **📢 Painel de Campanhas:** Visualização e gerenciamento de campanhas ativas da ONG.
- **🤝 Sistema de Associação:** Funcionalidade para que visitantes possam se associar à instituição.
- **💳 Pagamentos Online:** Processamento de doações avulsas e gerenciamento de assinaturas mensais para os sócios.
- **♿ Acessibilidade e Responsividade:** Design totalmente responsivo adaptado a múltiplos dispositivos e em conformidade com melhorias de acessibilidade.

---

## 🛠️ Tecnologias Utilizadas

- **Frontend:** React.js, JavaScript, Tailwind CSS, Django Template
- **Backend:** Python, Django framework
- **Banco de Dados:** PostgreSQL
- **Versionamento:** Git & GitHub
